## Enterprise Features

Note: The enterprise edition was cancelled. All features are now part of the open-source version.

Features originally planned for enterprise that are now in OSS:
- Advanced discovery (SSH, SNMP)
- Scan profiles
- Scheduled scans
- Credential management
- All IPAM features
